# 🤖 AI聚合助手

一款支持 **DeepSeek** 和 **智谱GLM** 的安卓AI聊天应用，采用HTML/CSS/JS原生开发，通过WebToApp打包。

## ✨ 功能特性

- 🤖 双AI模型切换（DeepSeek / 智谱GLM）
- 🎤 语音输入支持
- 🌙 深色模式
- 📋 一键复制AI回复
- 📜 本地历史记录（最多30条）
- 🎨 莫奈风格毛玻璃UI
